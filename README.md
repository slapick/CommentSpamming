# CommentSpamming
Вы можете накрутить комментарии под записью

```js
{
    "token": "045d9669b709b2431b516asd4asd56546hj8gdsf854df65g5fd5465b8d1996bfdcd5484d54fg5df6563414236b9ba3fab5", // Ваш токен
    "owner_id": "123456789", // id владельца записи
    "post_id": "111", // id записи
    "text": [ // Текст комментария
        "anything",
        "anything else"
        ],
    "number": 100, // Количество накручиваемых комментарием
    "delay": 5 // Задержка между комментариями в секундах
}
```
